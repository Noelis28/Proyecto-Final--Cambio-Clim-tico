# Proyecto-Final--Cambio-Clim-tico

Mi idea es: realizar una página Web sencilla, utilizando de idea o de base (dependiendo del tiempo), la página web realizada sobre cuánta energía utilizabamos en nuestro hogar.

Mi página Web se llamaría: EcoHuella

Funciones de la página Web:
    1. Calculadora ecológica, con preguntas sencillas:
    ¿Usas transporte público o carro?
    ¿Cuántas horas usas electricidad al día?
    ¿Consumos plástico con frecuencia?
 Con estas preguntas esperoq que la página web pueda estimar el impacto ambiental o la huella de carbono de las personas (bajo, medio o alto).
    2. Una sección de recomendaciones personalizadas:
    Según los resultados: 
    a. Podrían darse consejos para reducir emisiones
    b. Hábitos más sostenibles
    c. Cambios pequeños que se podrían adaptar pero que son efectivos.
    3. A su vez me gustaria implementar la síntesis de voz para que sea más dinámico, que pueda leer los resultados, animar al usuario o que pueda dar los consejos y recomendaciones hablados.

Para esto utilizaría el framework de flask
